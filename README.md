# Techy (Made by Ahmed Hamdy)

A frontend e-commerce website for tech products. Built with vanilla JavaScript, HTML5, and CSS3.

## Live Demo

**[https://techystore-ahmedhamdy.netlify.app/](https://techystore-ahmedhamdy.netlify.app/)**

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6 modules)
- LocalStorage (no backend)

## Features

- **Home** – Product catalog with search and filters (category, price)
- **Product Details** – View product info and add to cart
- **Cart** – Manage quantities, view total, checkout (Buy Now)
- **User Auth** – Register, login, logout
- **Profile** – View account info (requires login)
- **Contact** – Contact form and company info

## Running Locally

1. Clone the repository
2. Serve the project with a local server (e.g. `npx serve .` or VS Code Live Server)
3. Open the URL in your browser
